# 🔐 Binary & Unicode Explorer

A specialized **Python** tool built with **Streamlit** to visualize how human language is translated into machine-level data (0s and 1s).

## 🚀 Purpose
I created this tool to explore the underlying logic of my **MacBook M1 Air**. It serves as a "digital microscope" to see the binary representation of characters, especially comparing English ASCII and Chinese Unicode (like my name, **刘炜**).

## 🧪 What you can do:
- **Encode:** Type any string to see its **Binary (0b...)** and **Hexadecimal (0x...)** values.
- **Decode:** Paste binary strings to retrieve the original text.
- **Hardware Insight:** Understand how the CPU processes high and low voltage signals to represent data.

## 🛠️ Technology
- **Language:** Python 3.10+
- **Framework:** Streamlit
- **Concepts:** UTF-8 Encoding, Bitwise logic, Memory Management.
